# vrml
VRML置き場
個人的なVRMLファイルの置き場です。
GitHubの練習のために作りました。
プルリクエスト歓迎です。
何の役にもたたないかもしれませんが。

VRMLは3Dのオブジェクトを表現する言語です（多分）。 
VRML(拡張子 .wrl)を見るには 
Cortona 3D Viewer というActive Xのプラグインを入れて IE11で開いてください。
http://www.cortona3d.com/
